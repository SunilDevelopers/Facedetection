# Face Detection Attendance System

A Face Detection and Attendance Management System built using Python, Flask, OpenCV, and SQLite.

## Features

* Face Registration
* Face Recognition
* Attendance Tracking
* SQLite Database Storage
* REST API Support
* Easy Integration with Android Applications

## Technologies Used

* Python
* Flask
* OpenCV
* face_recognition
* SQLite

## Project Structure

* `app.py` - Main Flask API
* `attendance.db` - Database
* `faces/` - Registered face images

## Usage

1. Install required Python packages.
2. Run:

   ```
   python app.py
   ```
3. Access the API from your Android application.
